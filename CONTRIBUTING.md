# Contributing

Thanks for your interest in improving Forest Ash Theme.

## Ways to Contribute

- Report bugs or visual inconsistencies.
- Suggest color palette improvements.
- Propose new theme variants.
- Improve docs and examples.

## Before You Start

- Search existing issues to avoid duplicates.
- Open an issue first for larger changes so direction can be aligned.

## Development Setup

1. Fork and clone the repository.
2. Install dependencies:

   npm install

3. Open the project in VS Code.
4. Press F5 to launch an Extension Development Host.
5. Select a Forest Ash theme variant and verify your changes.

## Theme Changes

When adjusting token colors or UI colors:

- Keep contrast readable for long coding sessions.
- Test in common languages (JavaScript/TypeScript, JSON, Markdown).
- Check both dark and light variants when relevant.
- Avoid changing unrelated theme files in one pull request.

## Pull Request Guidelines

- Keep PRs focused and small.
- Use clear commit messages.
- Add screenshots when UI or color changes are involved.
- Reference the related issue number when possible.

## Issue Reporting Template

Please include:

- Theme variant name.
- VS Code version.
- OS version.
- Language/file type.
- A screenshot showing the issue.
- Steps to reproduce.

## Code Style

- Follow existing JSON structure and key ordering patterns.
- Keep formatting consistent with the rest of the repository.
