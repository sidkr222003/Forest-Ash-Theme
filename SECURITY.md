# Security Policy

## Supported Versions

Security fixes are provided for the latest published version of this extension.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

Preferred method:

- Use GitHub private vulnerability reporting for this repository (Security tab).

Alternative method:

- Open a private contact request through the repository owner profile on GitHub,
  and include "Security Report" in the subject.

Please include:

- Description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Suggested fix (if available).

You can expect an initial response within 7 days. After validation, I will work
on a fix and coordinate responsible disclosure timelines.

Please do not open public issues for unpatched vulnerabilities.
